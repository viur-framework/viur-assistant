ViUR Assistant is a powerful Python module designed to integrate AI assistant
capabilities seamlessly into your `ViUR <https://www.viur.dev>`_ projects.
It simplifies the usage of OpenAI and Anthropic APIs with easy
configuration and modular design.

Quick Links
-----------

- `GitHub Repository <https://github.com/viur-framework/viur-assistant>`_
- `PyPi <https://pypi.org/project/viur-assistant/>`_


Dependencies
------------

- `ViUR Core <https://core.docs.viur.dev/en/stable/>`_
- `OpenAI API <https://openai.com/api/>`_
- `Anthropic API <https://www.anthropic.com>`_
